Инструкция: Apps Script

  1. Таблица → Расширения → Apps Script
  2. Вставь код из mafia_apps_script.js → Сохрани (Ctrl+S)
  3. Развернуть → Новое развёртывание → Веб-приложение
  4. Выполнять как: Я | Доступ: Все (анонимные)
  5. Скопируй URL → вставь выше → Сохранить
